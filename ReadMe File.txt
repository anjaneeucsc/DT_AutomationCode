Pre requests:
1. java 8 or higher version should be installed 
2. Install maven and git

Steps to run project==========
1. open command prompt and go to the path where you want to keep the project. 
2. Enter below command in command prompt to clone the project.
git clone https://github.com/anjaneeucsc/DT_AutomationCode.git
3. Navigate to the DT_AutomationCode folder in command prompt.
4. Type below command
mvn clean verify
5.let all the scenarios to be executed.( You can see 'BUILD SUCCESS' once after the execution. )
6.Go inside DT_AutomationCode/singtel-assesment-todos/target/site/serenity
7.Open index.html to view report
